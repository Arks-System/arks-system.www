Arks-System Tool

Cette version est actuellement une version bêta. N'hésitez pas à nous faire part des problèmes rencontré sur le GitHub ou sur #support de notre Discord.

Cet outil est un lanceur alternatif pour PSO2 et permet aussi de le mettre à jour depuis les serveurs officiels en plus de s'assurer que les droits des fichiers soient correctement paramétrés.
Il permet aussi de traduire une partie du jeu en anglais.

Veuillez noter que le support du mirroir Arks-System a été désactivé car cette fonctionnalité n'est plus supporté.
